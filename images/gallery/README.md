# Gallery Images

This directory contains all images for the Buy Junk Car Miami photo gallery.

## Directory Structure

- `before-after/` - Before and after photos of junk car removals
- `reviews/` - Customer testimonial photos
- `team/` - Team member photos  
- `facility/` - Photos of our facility, tow trucks, and operations
- `customers/` - Additional customer photos

## Image Requirements

- **Format**: JPG or PNG
- **Size**: Recommended 800x600px minimum
- **Quality**: High resolution for professional appearance
- **Naming**: Use descriptive names like `honda-civic-before.jpg`

## Usage

Images are used in:
- `/gallery/` - Main gallery page with slideshow
- Homepage gallery preview section
- Lightbox modal displays
- Customer testimonial sections

All images should be optimized for web and include proper alt text for accessibility.
